## [Unreleased]

### Added
- Initial open source release
- Multi-tenant authentication with passkey, TOTP, and magic link support
- OAuth 2.0 / OIDC server with PKCE support
- JWT signing with automatic key rotation
- Webhook event system with retry logic
- Audit logging with retention policies
- IP policy enforcement (allowlist/blocklist)
- GDPR data export and deletion endpoints
- Personal Access Tokens (PAT) for API authentication
